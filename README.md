# snakeGame
# 🐍 Vibecoded Snake Game

Welcome to my completely **vibecoded** take on the classic Snake game! 

This project started as a fun experiment to see how much "juice" and modern feel I could inject into a simple grid-based game using pure HTML, CSS, and JavaScript. My intention is to build something that feels incredible to play, looks super slick, and requires absolutely zero setup to run.

And trust me—**this game will only keep getting better.** 🚀

---

## ✨ Features

This isn't your standard, clunky 1990s snake. It's packed with modern visual effects, physics, and quality-of-life features:

* 🏎️ **Smooth Movement Engine:** Instead of instantly teleporting block-by-block, the snake moves with buttery smooth, interpolated frame-by-frame rendering.
* 💥 **Juicy Visual Effects:** * **GTA-Style Death:** Hit a wall? Get a dramatic, cinematic `WASTED` screen.
    * **Bounceback Physics:** The snake violently recoils backwards upon impact.
    * **Particle Systems:** Hit a mine? Massive fire and smoke explosion. Bite yourself? Red blood splatters that permanently stain the ground.
* 💀 **Dynamic Difficulty:** * **Easy:** Slower pace, traditional gameplay.
    * **Hard:** Faster speed and deadly, pulsating neon mines that spawn dynamically. 
* 🎨 **Customization:** Choose between 8 different vibrant gradient color themes for your snake (Blue, Red, Purple, Green, Orange, Pink, Yellow, Cyan).
* 🏆 **Hassle-Free Leaderboard:** A built-in, tabbed leaderboard (Easy vs. Hard) that tracks your top 5 high scores locally using `localStorage`. No backend databases, no Supabase keys, no API limits—it just works out of the box.
* 📱 **Mobile Responsive:** Automatic touch controls (on-screen D-pad) appear when playing on smaller screens or mobile devices.
