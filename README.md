A modern, pure-JS take on the classic. Built on pure vibes, and it will only keep getting better.

## Features

- **Smooth Engine:** Interpolated frame-by-frame movement instead of clunky grid snapping.
- **Juicy Physics:** Bounceback recoil, fire particle explosions, and permanent blood splatters.
- **WASTED:** GTA-style cinematic death screens.
- **Modes:** Easy (classic) and Hard (faster pace + deadly neon mines).
- **Customization:** 8 different gradient color themes.
- **Leaderboards:** Zero-setup local high score tracking.
- **Responsive:** Auto-enabled D-pad for mobile screens.

## Play

No build steps. No dependencies. 

Just open `index.html` in any browser and use **WASD** or **Arrows** to move.
